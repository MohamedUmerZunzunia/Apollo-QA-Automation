# Apollo-QA-Automation

## Overview
This project contains Selenium WebDriver automation tests written in Java for the Practice Software Testing website.

Website:
https://practicesoftwaretesting.com

## Technologies Used
- Java
- Selenium WebDriver
- IntelliJ IDEA
- ChromeDriver
- Git & GitHub

## Test Scenarios

### 1. Registration Test
- Open registration page
- Fill customer details
- Submit registration form

### 2. Login Test
- Open login page
- Enter registered credentials
- Verify successful login

### 3. Checkout Test
- Open Practice Software Testing website
- Select Combination Pliers
- Add product to cart
- Open cart
- Proceed to checkout
- Login with registered account
- Enter billing address
- Select payment method
- Confirm order
